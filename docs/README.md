# Documentation

This directory contains the documentation for DS-pyVDC-API.

## Building the Documentation

To build the documentation locally:

```bash
pip install -e ".[docs]"
cd docs
make html
```

The HTML documentation will be generated in `docs/_build/html/`.

## Documentation Structure

- **Getting Started**: Installation and quick start guide
- **API Reference**: Detailed API documentation
- **Examples**: Usage examples and tutorials
- **Contributing**: Guidelines for contributors

## Using Sphinx

This project uses Sphinx for documentation generation. The documentation is written in reStructuredText (reST) format.
